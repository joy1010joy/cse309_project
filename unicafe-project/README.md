# UniCafe - AI-Powered Cafe Pre-Order System

## Features
- Menu browsing with AI recommendations
- Pre-order with pickup time
- Order tracking (pending/confirmed/ready)
- Admin dashboard with analytics
- AI chat assistant
- AI business insights for admin

## Tech Stack
- Backend: FastAPI (Python)
- Database: Firebase Firestore
- AI: Google Gemini API
- Frontend: HTML, Tailwind CSS, JavaScript

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Set environment variables in `.env`
3. Add Firebase service account JSON
4. Run: `uvicorn main:app --reload`
5. Open http://localhost:8000
